# tmux mode굿(key default라 세팅 따로 필요없음)
- 극한의 환경에서 쓸만함

```bash
# 새창 추가
Ctrl+ b + c

# vs 창추가 (right)
Ctrl + b + %

# sp 창추가 (down)
Ctrl + b + "

# 창 전환(Previous)
Ctrl + b + p

# 창 전환(Next)
Ctrl + b + n 
```

# 많이 쓰는거

```bash
# 창 만들기
<c-b> c

# 가운데 터미널창 조그만하게 띄우기
<c-p> alt+f d
or
<c-p> e

# 창 전환
Alt 화살표
or
Alt h or l


# vs로 창 나누기
<c-p> n


# 키 잠그기 key lock
<c-g>

# vim모드로 보기, 창에 보이는거 파일로 만들수도 있음
<c-s> e

```
